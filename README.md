# Hello World

一个简单的HTML页面，展示"Hello"问候语。

## 功能特性

- 🎨 现代化渐变背景设计
- ✨ 平滑的淡入动画效果
- 📱 响应式布局，适配各种设备
- 🌈 优雅的阴影效果

## 技术栈

- HTML5
- CSS3
- 动画和过渡效果

## 快速开始

1. 克隆仓库：
   ```bash
   git clone https://github.com/gavinli1245gm/hello.git
   ```

2. 进入项目目录：
   ```bash
   cd hello
   ```

3. 用浏览器打开 `index.html` 文件

## 在线预览

如果仓库启用了GitHub Pages，可以通过以下链接访问：
```
https://gavinli1245gm.github.io/hello/
```
